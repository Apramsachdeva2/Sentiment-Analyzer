What Is Sentiment Analysis


Sentiment Analysis can be described as the process of computationally identifying and categorizing opinions expressed in a piece of text, especially in order to determine whether the writer's attitude towards a particular topic and giving a label as positive negative or nuetral



Twitter is a social network where people post short, 140-character, status messages called tweets,because tweets are sent out continuously, Twitter can prove to be a great way to figure out how people feel about current events, and their tweets on a topic can be used to analyse their sentiment about the topic.

To Analyse the Sentiments

First streaming tweets from twitter API and filtering out the tweets relevant to the topic

Analysing each and every tweet using natural language processing and giving them a label

And then prepearing a report from the number of positive/negative/nuetral labels given to the tweets
